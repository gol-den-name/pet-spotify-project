# Spotify Clone (personal project)

A mini Spotify UI clone built using **HTML, CSS, and JavaScript**.  
Includes track list, play/pause button, highlight effect, and random song playback.

## Features
✅ Click to play any song  (managed by html and js file)
✅ Auto highlights currently playing track  (js file)
✅ Play/pause button  (js file)
✅ Random song if no track selected  (js file)
✅ Fully responsive layout  (Html and CSS file)
